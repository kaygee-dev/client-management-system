# R4MZ Media Quote Generator 💼📸

This is a work-in-progress JavaScript project that generates quotes for a photography and videography business.

The system calculates pricing based on service type, hours worked, travel requirements, and applies discounts where applicable.

---

## 🚧 Current Features (Phase 1)

- Pricing rules for:
  - Photography (R500/hour)
  - Videography (R800/hour)
  - Both services (R1200/hour)

- Travel fee calculation (R250 if required)
- Discount system (10% off for totals ≥ R3000)
- Quote generation for multiple clients
- Modular functions for calculations

---

## 🧠 What this project is about

This project is being built to simulate a real-world freelance media pricing system.  
It helps me practice:

- JavaScript functions
- Arrays and objects
- Loops
- Business logic building
- Code structure and reusability
